# globalhubs.top — open datasets

Open, well-documented datasets of the world's largest **freight** transport hubs
and operators. Passengers are out of scope. Site: https://globalhubs.top · License: CC0 1.0.

**Hubs** — the largest freight places:

| Dataset | Folder | Ranked by |
|---|---|---|
| Maritime — largest container ports | [`maritime/`](./maritime) | container throughput (TEU) |
| Aviation — busiest cargo airports | [`aviation/`](./aviation) | air cargo (tonnes) |
| Land — largest inland freight hubs | [`land/`](./land) | container throughput (TEU) |

**Operators** — the largest freight carriers:

| Dataset | Folder | Ranked by |
|---|---|---|
| Container shipping lines | [`container-shipping-lines/`](./container-shipping-lines) | operated fleet capacity (TEU) |
| Air cargo carriers | [`air-cargo-carriers/`](./air-cargo-carriers) | cargo tonne-km (CTK) |
| Freight railways | [`freight-railways/`](./freight-railways) | net freight tonne-km |

Each folder holds a CSV + Parquet and a README with the schema, methodology and caveats.
